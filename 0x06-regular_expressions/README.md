# 0x06. Regular expression
* Allowed editors: vi, vim, emacs
* All your files will be interpreted on Ubuntu 20.04 LTS
* All your files should end with a new line
* A README.md file, at the root of the folder of the project, is mandatory
* All your Bash script files must be executable
* The first line of all your Bash scripts should be exactly #!/usr/bin/env ruby
* All your regex must be built for the Oniguruma library\w is equivalent to [0-9a-zA-Z_]

# Information
* <code> \d </code> is the same as <b>[0-9]</b>
* <code> \s </code> matches white space (tabs, regular space, newline)
* <code> \W </code> anything that’s not in <b>[0-9a-zA-Z_]</b>
* <code> \D </code> anything that’s not a number
* <code> \S </code> anything that’s not a space
* <b> [0-9] </b> matches any number from 0 to 9
* <b> [a-z] </b>  matches any letter from a to z (no caps)
* <b> [^a-z] </b> negated range
